# SubTrack

A React-based subscription management dashboard that helps users track recurring subscriptions, monitor spending, and visualize expenses by category.

## Features

- Add, Edit, and Delete subscriptions
- Monthly and Annual spending analytics
- Upcoming renewal tracking
- Category-based expense breakdown
- Interactive Pie Chart visualization
- Local Storage persistence
- Responsive dashboard UI

## Tech Stack

- React
- JavaScript
- CSS
- Chart.js
- localStorage

## Installation

npm install

npm run dev
